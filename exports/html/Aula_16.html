Anlise de dados

Agora que os dados j foram tratados, podemos iniciar a Anlise de dados propriamente dita. Que tal comearmos analisar a taxa de cancelamento da empresa, supondo que o nosso objetivo seja diminuir essa taxa. Para isso, vamos ter que verificar qual essa taxa e de onde ela vem:

In[]:
display(tabela['cancelou'].value_counts())

Pronto! Temos um nmero. A linha do 1.0 representa quem cancelou, enquanto a linha do 0.0 representa quem no cancelou. Mas esse nmero representa qual porcentagem do total? Vamos analisar:

In[]:
display(tabela['cancelou'].value_counts(normalize=True).map('{:.2%}'.format))

cancelou

O .map() atribui lista uma operao, que nesse caso a transformao dos valores em porcentagem.

Veja que, de acordo com a nossa anlsie, mais da metade dos clientes cancelou o servio. O prximo passo analisar e tentar entender de onde vem uma taxa to alta de cancelamento.

Grficos

Uma das melhores formas de se analisar dados atravs de grficos, pois fica muito mais fcil visualizar os dados e obter as informaes necessrias.

O Python extremamente rico em bibliotecas que geram grficos, e disso definitivamente no temos do que reclamar. Para o nosso caso especfico, iremos instalar e importar a biblioteca plotly.express, usando, claro, o comando pip install plotly.

OBS: Caso tenha problemas na visualizao dos grficos depois de instalar a biblioteca, volte ao terminal e escreva pip install nbformat. Se mesmo aps isso no funcionar, reinicie o VSCode e rode todos os cdigos do notebook novamente.

Vamos usar o plotly para criar um histograma com cada uma das colunas, assim podemos analisar cada uma das informaes e verificar como elas se comportam em relao aos cancelamentos da empresa.

In[]:
import plotly.express as px

for coluna in tabela.columns:
    grafico = px.histogram(tabela, x=coluna, color="cancelou", width=600)
    grafico.show()

Observe que o cdigo gerou 11 grficos: um para cada coluna. claro que teremos que analisar cada um deles, mas vamos dar um spoiler: observe os grficos dias_atraso e ligacoes_callcenter:

Grfico das Ligaes ao Call Center
